1. 適用範囲

本ポリシーは、Chrome拡張機能「シゴデキ」（以下「本拡張機能」）の利用において、 本拡張機能が取り扱う情報の範囲・利用目的・保存期間等を定めるものです。

2. 取得・取り扱う情報

2.1 ユーザーが入力する情報（任意）

  • 問い合わせテンプレート文、設定値（例：動作モード、除外設定 等）

2.2 本拡張機能がGoogleマップ上で参照する情報

本拡張機能は、ユーザーが閲覧中の Google マップページ上で、サイドパネル表示や支援機能のために 事業所情報（例：店舗名、WebサイトURL、地図上のリンク等）を参照します。 これらはユーザーが明示的に本拡張機能を操作した範囲で利用します。

2.3 端末内（ローカル）への保存

本拡張機能は、利便性向上のため、Chrome の拡張機能ストレージ（chrome.storage）に 2.1の情報等を端末内に保存する場合があります。

2.4 サーバーへの送信（機能を有効化している場合）

本拡張機能が、問い合わせ支援・処理のためのバックエンド機能（例：クラウド上のAPI）と連携する場合、 次の情報の一部を送信することがあります（送信の有無・範囲は実装と設定に依存します）。

  • ユーザーが操作対象として選択した事業所情報（例：店舗名、URL 等）
  • 問い合わせフォーム解析に必要なページURL、フォーム構造（HTML断片等）
  • 処理結果（成功/失敗/エラー理由等）
  • 不正防止・障害解析のためのログ情報（例：アクセス日時、リクエスト識別子等）

なお、個人情報・機微情報を取り扱う場合は、暗号化通信（HTTPS）等により安全に取り扱います。

3. 利用目的

  • 本拡張機能の提供（事業所情報の表示、問い合わせ作業支援 等）
  • 設定の保持、利便性向上
  • 品質改善、障害対応、不正利用防止
  • ユーザーからの問い合わせ対応（必要な場合）

4. 第三者提供・外部送信

当方は、法令に基づく場合等を除き、ユーザーデータを第三者へ販売・貸与することはありません。 ただし、提供機能の実現に必要な範囲で、クラウド事業者・ホスティング事業者等の委託先に 取り扱いを委託する場合があります（例：サーバー運用、ログ保管）。

5. 保存期間

  • 端末内（ローカル）に保存された設定等：ユーザーが削除するか、本拡張機能をアンインストールするまで
  • サーバー側ログ等（ある場合）：[例：最大30日] を目安に、目的達成に必要な期間のみ

6. セキュリティ

当方は、情報の漏えい・改ざん・不正アクセス等を防止するため、合理的な安全管理措置を講じます。 個人または機微情報を取り扱う場合、暗号化通信（HTTPS）など現代的な暗号技術を用いて送受信します。

7. ユーザーの選択肢

  • 端末内の保存情報は、拡張機能の設定画面やブラウザの機能で削除できます
  • 本拡張機能を無効化/アンインストールすることで、動作を停止できます
  • サーバー側に保存された情報の削除依頼（ある場合）：上記連絡先へご連絡ください

8. 改定

当方は、必要に応じて本ポリシーを改定することがあります。重要な変更がある場合は、適切な方法で告知します。

9.（該当する場合）Limited Use に関する表明

Google API から受領した情報を利用する場合、当該情報の利用は Chrome Web Store User Data Policy の Limited Use 要件に従います。
